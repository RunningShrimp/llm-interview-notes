## 🌟 最新优化

本网站已经过全面优化，提供更好的阅读体验：

### ✨ 主要特性

- **📑 智能目录导航**：左侧自动生成的目录，支持快速跳转和当前位置高亮
- **🎨 现代化设计**：优雅的排版、舒适的字体大小和行高
- **💻 代码块增强**：语法高亮、一键复制、行号显示
- **🌓 深色模式**：支持系统偏好检测和手动切换（点击右上角按钮或按 `t` 键）
- **📱 响应式设计**：完美适配桌面、平板和手机
- **⬆️ 快速返回**：右下角的返回顶部按钮（或按 `ESC` 键）
- **🎯 可访问性**：符合无障碍标准，支持键盘导航

### 📖 使用指南

- **目录导航**：点击左侧目录快速跳转到任意章节
- **主题切换**：点击右上角的 🌙/☀️ 按钮或按键盘 `t` 键
- **返回顶部**：点击右下角的 ↑ 按钮或按 `ESC` 键
- **复制代码**：悬停在代码块上会显示复制按钮
- **键盘快捷键**：
  - `t` - 切换明暗主题
  - `ESC` - 返回页面顶部
